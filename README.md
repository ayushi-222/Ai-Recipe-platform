# 🍳 AI Recipe Platform

A modern Full-Stack AI-powered Recipe Platform built with Next.js, Tailwind CSS, Shadcn UI, Neon PostgreSQL, Strapi CMS, and Arcjet security. The platform allows users to discover, create, and manage recipes with the help of AI-generated content and intelligent recommendations.

## ✨ Features

* 🤖 AI-powered recipe generation and suggestions
* 🔐 Secure authentication and application protection with Arcjet
* 📖 Create, manage, and browse recipes
* 🎨 Modern and responsive UI built with Tailwind CSS and Shadcn UI
* ⚡ Fast server-side rendering and routing with Next.js
* 🗄️ PostgreSQL database powered by Neon DB
* 📝 Content management using Strapi CMS
* 🔍 Search and explore recipes effortlessly
* 📱 Mobile-friendly and responsive design

## 🛠️ Tech Stack

### Frontend

* Next.js
* React
* Tailwind CSS
* Shadcn UI

### Backend & Database

* Strapi CMS
* Neon PostgreSQL

### Security

* Arcjet
